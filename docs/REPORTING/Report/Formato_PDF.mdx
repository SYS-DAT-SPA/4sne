# Formato PDF

![icona_pdf.jpg](/UFR/imgs/icona_pdf.jpg)

 Report PDF In questo tipo di report l'output è un documento in formato **PDF**.  Le colonne del foglio Excel sono i campi di selezione indicati nella [**Query SQL**](/UFR/imgs/Report/Gestione_Report_Comandi_SQL) di estrazione dati.  ---

Attivazione Questo tipo di report si abilita selezionando l'opzione **Formato PDF** nel campo di selezione del **Formato Report** della pagina di configurazione, come mostrato sotto.  

![formato_pdf2.jpg](/UFR/imgs/formato_pdf2.jpg)

 

Configurazione del modulo PDF Di norma questo tipo di report necessità di una stampa Microsoft **Crystal Report**. La stampa va agganciata al report tramite il campo Nome report, come mostrato sotto. 

![configurazione_pdf.jpg](/UFR/imgs/configurazione_pdf.jpg)

 

### Agganciare stampa Crystal report Per inserire la stampa creata in Crystal report occorre premere sul pulsante 

![pulsante_più.jpg](/UFR/imgs/pulsante_più.jpg)

 Il sistema aprirà una finestra nella quale trascinare il modulo Crystal. 

![inserimento_modulo_crystal.jpg](/UFR/imgs/inserimento_modulo_crystal.jpg)

 Confermare premendo sul pulsante Conferma.  La stampa Crystal report può essere variata utilizzando il pulsante
