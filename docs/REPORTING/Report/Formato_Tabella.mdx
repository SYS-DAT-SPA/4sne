# Formato Tabella Interattiva

![tabella.jpg](/UFR/imgs/tabella.jpg)

Report Tabella Interattiva In questo tipo di report l'output è una  tabella a video.  

![tabella_2.jpg](/UFR/imgs/tabella_2.jpg)

  Le colonne della tabella sono i campi di selezione indicati nella [**Query SQL**](/UFR/imgs/Report/Gestione_Report_Comandi_SQL) di estrazione dati.  ---

Attivazione Questo tipo di report si abilita selezionando l'opzione **Formato Tabella** nel campo di selezione del **Formato Report** della pagina di configurazione, come mostrato sotto.  

![formato_tabella.jpg](/UFR/imgs/formato_tabella.jpg)

 

Ordinamento colonne L'ordinamento relativo alla comparsa delle colonne è gestito attraverso il tab **Predefinito**. 

![predefinito.jpg](/UFR/imgs/predefinito.jpg)

 Aprendo questo tab il sistema espone tutti i campi recuperabili dal comando SQL di estrazione dati. Con la colonna **Etichetta** è possibile attribuire uno specifico noma alla colonna: Mentre con il campo **Ordinameto** è possibile assegnare un ordine di comparsa alle colonne della tabella. 

![predefinito2.jpg](/UFR/imgs/predefinito2.jpg)

 

Filtri Rimangono valide tutte le regole e considerazioni per la gestione dei filtri presenti nella parte di [**Configurazione Report**](/UFR/imgs/Report/UFR_Cinfigurazione_Report).
