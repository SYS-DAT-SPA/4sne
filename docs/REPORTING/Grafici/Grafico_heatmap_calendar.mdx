# Grafico Heatmap Calendar  

Ogni record dovrà presentare un campo Value per il valore ed una campo Date per la data in formato "dd/mm/yyyy". Si può inserire il campo Legend per la legenda tooltip.

 Selezione tipo del grafico Questo tipo di grafico si attiva selezionando l'opzione "Formato Calendario HeatMap" nel campo formato report.
