# Singolo Valore 

Con questi tipo di grafico viene visualizzata una singola "mattonella" nella quale compare un titolo ed un valore di riferimento.   

![immagine_2022-11-22_143356.png](/UFR/imgs/immagine_2022-11-22_143356.png)

  

 Selezione tipo del grafico Questo tipo di grafico si attiva selezionando l'opzione "Formato Singolo Valore" nel campo formato report.  

![formato_grafico_valore.jpg](/UFR/imgs/formato_grafico_valore.jpg)

  

 Definizione SQL La query SQL deve fornire il valore chiamato **Title** ed il sottotitolo come campo **SubTitle**. Di fatto la Query SQL deve restituire solo due valori.  **Esempio 1** `SELECT 'mc' as SubTitle, value as Title... `  **Esempio 2** `SELECT 'cartons' as SubTitle, value as Title... `
