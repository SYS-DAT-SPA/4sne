# Grafico a Torta e Collegamento

In questo tipo di grafico è possibile aggancoiare un report "figlio" per eseguire un **Drill-Down** sui dati.  

![drilldown.jpg](/UFR/imgs/drilldown.jpg)

   Per quanto concerne la struttura della **Query SQL** rimangono valide tutte le specifiche discusse per il [**Grafico a Torta**](/UFR/imgs/Report/Grafico_A_Torta) semplice. 

Drill-Down Per abilitare il drill-down verso un altro report è necessario compilare i campi di **Link** tra i due report.  L'immagine sotto riporta sotto **Link** il riferimento al report figlio da agganciare ed il campo di filtro da utilizzare per far ereditare eventuali filtri al report collegato. 

![grafico_figlio.jpg](/UFR/imgs/grafico_figlio.jpg)

  **Attenzione!** Il report di link deve avere al suo interno definiti dei filtri per poter ereditare dati dal report principale.
