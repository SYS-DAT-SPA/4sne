# Grafico Radar 

Ogni record forma una serie ovvero un entità espressa tramite un numero predefinito di attributi (chiamate dimensioni o categorie). Ogni campo del record può essere una dimensione e deve perciò avere un valore numerico. Il nome delle dimensioni da rappresentare deve essere inserito nel box "Categoria" come per il barchart. 

![immagine_2022-11-22_143426.png](/UFR/imgs/immagine_2022-11-22_143426.png)

 

 Selezione tipo del grafico Questo tipo di grafico si attiva selezionando l'opzione "Formato Singolo Valore" nel campo formato report.
