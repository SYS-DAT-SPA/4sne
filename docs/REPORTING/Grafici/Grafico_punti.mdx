# Grafico a Punti  

Similmente al grafico a linee, sulle ascisse devono comparire valori categorial oppure numerici nel campo chiamato x, mentre i valori di ordinata sono da fornire nel campo y.  "Inserire screen" 

 Selezione tipo del grafico Questo tipo di grafico si attiva selezionando l'opzione "Formato Grafico a Punti" nel campo formato report.
