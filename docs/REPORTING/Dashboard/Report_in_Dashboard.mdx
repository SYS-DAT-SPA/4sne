### Gestione Azioni e Menu. 

Tutti report o le dashboard configurate nello UFR possono essere utilizzate dagli utenti finali sotto forma di azioni a menu. Tali azioni e menu possono essere configurati e personalizzati in base alle specifiche esigenze aziendali. Le informazioni per gestire tale configurazione sono consultabili nel capitolo Menu/Azioni. 

### Gestione Utenti 

Per accedere al sistema di reportistica ed in generale su Next occorre avere un utente e password abilitati. 

Per creare un utente consultare il capitolo Gestione Utenti nel quale sono riportate tutte informazioni necessarie per gestire un utente.
