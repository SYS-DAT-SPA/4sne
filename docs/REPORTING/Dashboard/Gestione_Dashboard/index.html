Gestione Dashboard

La Dashboard è un componente del tool che consente di aggregare e visualizzare dati provenienti da più report. I report presenti nella dashboard vengono eseguiti in maniera simultanea. I dati possono essere esaminati e parzializzati applicando filtri. Una dashboard è lo strumento ideale per - Aggregare analisi di dati che hanno una correlazione - Identificare tendenze - Misurare e monitorare fenomeni legati alle attività aziendali quotidiane --- In questa sezione troverai le informazioni per poter creare e configirare una Dashboard.

Sezione 1

Gestione Dashboard La Dashboard è un componente del tool che consente di aggregare e visualizzare dati provenienti da più report. I report presenti nella dashboard vengono eseguiti in maniera simultanea. I dati possono essere esaminati e parzializzati applicando filtri. Una dashboard è lo strumento ideale per - Aggregare analisi di dati che hanno una correlazione - Identificare tendenze - Misurare e monitorare fenomeni legati alle attività aziendali quotidiane --- Le dashboard sono configurabili attrverso la voce di menu: Configurazione > Configurazione Dashboard

In questo modulo troverai informazioni per: Avviare la creazione di una nuova dashboard. Entrare in modifica di una dashboard esistente. Gestire la configurazione di una dashboard. Eliminare una dashboard esistente. Copiare una dashboard esistente

Avviare la creazione di una nuova dashboard. Per aggiungere una nuova dashboard occorre:

  1. Selezionare la voce di menù Configurazione > Configurazione dashboard come mostrato nella figura sotto.

  1. Premere sul pulsate Nuovo come indicato in giallo nella figura sotto, che compare nella pagina di elenco delle dashboard esistenti.

  1. Il sistema apre la pagina di configurazione del report nella quale andranno definiti tutti gli elementi necessari per la creazione della dashboard. Questa pagina è dettagliata nel capitolo (da fare) Configurazione di un Report.

Configurazione e manutenzione di una dashboard. In questa sezione troverai informazioni relative alla configurazione di una dashboard. La configurazione di una dashboard prevede una parte di definizione ed una parte di comportamento. La parte di definizione serve identificare la dashboard, mentre la parte comportamentale intoduce elementi quali: filtri, filtri di default, eventuale ordinamento dei report.

Dati generali - Identificazione dashboard Alla creazione di una dashboard occorre inserire un codice (non modificabile successivamente), ed una descrizione. Questi due elementi identificano la dashboard e la rendono ricercabile all'interno del sistema.

Definizione elenco reports Questa parte di configurazione serve ad inserire i Reports che devono essere visualizzanti all'interno della dashboard.

Per aggiungere un report all'elenco occorre: 1) Premere sul pulsante aggiungi.

  1. Il sistema apre un campo di selezione nel quale è possibile ricercare il report di interesse.

I report possono essere selezionati attraverso il menu a tendina.

  1. Selezionato il report di interesse occorre indicare in che posizione della dashboard deve essere visualizzato.

Sono disponibili queste opzioni. a) 1/3 - (1 di 3) Ossia il report selezionato fa parte di uno dei 3 che saranno visualizzati contemporaneamente nella medesima riga. Nell'esempio sotto sono stati selezionati 3 report che compariranno nella medesima riga della dashboard.

b) 1/2 - (1 di 2) Ossia il report selezionato fa parte di uno dei 2 che saranno visualizzati contemporaneamente nella medesima riga. Nell'esempio sotto sono stati selezionati 2 report che compariranno nella medesima riga della dashboard.

c) Full - Ossia il report sarà visualizzato in Full Screen. Pertanto nella medesima riga sarà presente solo il report selezionato.

Per aggiungere altri report ripetere le opreaiozni sopra.

Definizione Filtri Questa parte della configurazione serve per aggiungere dei filtri di selezione. Questi filtri saranno ereditati da tutti i report presenti nella dashboard purchè tali report facciano esplicito riferimento ai campi di selezione inseriti.

Aggiunta di un filtro Per inserire un filtro premere sul pulsante aggiungi della barra dei filtri. Il sistema apre la finestra di definizione del filtro.​

Eliminare un filtro Per rimuovere un filtro dal report o dalla dashboard premere il pulsante​

in corrispondenza del filtro da rimuovere nell'elenco dei filtri.

Modificare un filtro Per modificare un filtro premere il pulsante​

in corrispondenza del filtro da gestire nell'elenco dei filtri.

Premuto il pulsante di modifica il sistema apre la finestra di gestione del filtro.

Salvataggio della dashborad. Per salvare la configurazione della dashboard premere sul pulsante Salva.

Come copiare una dashboard esistente. Per duplicare una dashboard esistente occorre:

  1. Selezionare la voce di menù Configurazione > Configurazione dashboard come mostrato nella figura sotto.

  1. Il sistema espone l'elenco delle dashboard esistenti. Individuare la dashboard che si intende copiare ed in corrispondenza di tale dashboard premere sul pulsante menu

come mostrato nella figura sotto.

  1. Il sistema apre il menù a tendina dal quale occorre premere sul pulsante Duplica

  1. Il sistema aprirà la pagina di gestione della nuova dashboard nella quale occorre attribuire un nuovo codice e relativa descrizione. La nuova dashboard è soggetta a tutte le regole di gestione descritte nel capitolo di Configurazione di un Dashboard
