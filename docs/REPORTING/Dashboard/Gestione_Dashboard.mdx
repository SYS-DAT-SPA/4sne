# Gestione Dashboard

![dashboard_icona.jpg](/UFR/imgs/dashboard_icona.jpg)

 
 La Dashboard è un componente del tool che consente di aggregare e visualizzare dati provenienti da più report.  I report presenti nella dashboard vengono eseguiti in maniera simultanea.  I dati possono essere esaminati e parzializzati applicando filtri.  Una dashboard è lo strumento ideale per - **Aggregare analisi di dati che hanno una correlazione** - **Identificare tendenze** - **Misurare e monitorare fenomeni legati alle attività aziendali quotidiane**   --- In questa sezione troverai le informazioni per poter [**creare e configirare una Dashboard**](/UFR/imgs/Dashboard/UFR_Creare_gestire_dashboard).

# Configurazione e manutenzione di una dashboard.  In questa sezione troverai informazioni relative alla configurazione di una dashboard.   La configurazione di una dashboard prevede una parte di definizione ed una parte di comportamento. La parte di definizione serve identificare la dashboard, mentre la parte comportamentale intoduce elementi quali: filtri, filtri di default, eventuale ordinamento dei report.  

![configurazione_dettagli_dashboard.jpg](/UFR/imgs/configurazione_dettagli_dashboard.jpg)



Dati generali - Identificazione dashboard  Alla creazione di una dashboard occorre inserire un codice (non modificabile successivamente), ed una descrizione. Questi due elementi identificano la dashboard e la rendono ricercabile all'interno del sistema.   

![ufr_report_codice.jpg](/UFR/imgs/ufr_report_codice.jpg)

 

Definizione elenco reports  Questa parte di configurazione serve ad inserire i [**Reports**](/UFR/imgs/Report/Gestione_Reports) che devono essere visualizzanti all'interno della dashboard.  

![dashboard_elenco_reports.jpg](/UFR/imgs/dashboard_elenco_reports.jpg)

 Per aggiungere un report all'elenco occorre: 1) Premere sul pulsante aggiungi. 

![dashboard_aggiungi.jpg](/UFR/imgs/dashboard_aggiungi.jpg)

 2) Il sistema apre un campo di selezione nel quale è possibile ricercare il report di interesse. 

![dashboard_aggiungi_2.jpg](/UFR/imgs/dashboard_aggiungi_2.jpg)

 I report possono essere selezionati attraverso il menu a tendina. 

![dashboard_elenco_report_selezionabili.png](/UFR/imgs/dashboard_elenco_report_selezionabili.png)

 3) Selezionato il report di interesse occorre indicare in che posizione della dashboard deve essere visualizzato. 

![dashboard_ordinamento.jpg](/UFR/imgs/dashboard_ordinamento.jpg)

 Sono disponibili queste opzioni. a) **1/3** - (1 di 3) Ossia il report selezionato fa parte di uno dei 3 che saranno visualizzati contemporaneamente nella medesima riga. Nell'esempio sotto sono stati selezionati 3 report che compariranno nella medesima riga della dashboard. 

![dashboard_ordinamento_3_elementi.jpg](/UFR/imgs/dashboard_ordinamento_3_elementi.jpg)

 b) **1/2** - (1 di 2) Ossia il report selezionato fa parte di uno dei 2 che saranno visualizzati contemporaneamente nella medesima riga. Nell'esempio sotto sono stati selezionati 2 report che compariranno nella medesima riga della dashboard. 

![dashboard_ordinamento_2_elementi.jpg](/UFR/imgs/dashboard_ordinamento_2_elementi.jpg)

 c) **Full** - Ossia il report sarà visualizzato in Full Screen. Pertanto nella medesima riga sarà presente solo il report selezionato. 

![dashboard_ordinamento_1_elemento.jpg](/UFR/imgs/dashboard_ordinamento_1_elemento.jpg)

  Per aggiungere altri report ripetere le opreaiozni sopra.
