### Guide for Ordine Cliente New

*Add the description here*

___

1\. Per prima cosa **fai clic su** sull'elemento **i.fa. fa-plus**.  
  
2\. **Seleziona** un'opzione dall'elemento a discesa **con data test è uguale a "pCdAna"** .  
  
3\. Subito dopo, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pCdStagion"** .  
  
4\. Successivamente, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pCdLinea"** .  
  
5\. Successivamente, **Seleziona** un'opzione dall'elemento a discesa **con test dati uguale a "pCdFlash"** .  
  
6\. In secondo luogo, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pCdColle"** .  
  
7\. Poco dopo, **fai clic su** sull'elemento **con i dati test è uguale a "RifOrdCli"**.  
  
8\. Subito dopo, **Digitare** **"riferimento ordine cliente "** nell'elemento **con dati test uguale a "RifOrdCli"**.  
  
9\. Successivamente, **fai clic su** sull'elemento con **ID " Nota"**.  
  
10\. Subito dopo, **Digita** **"note ordine se esististono"** nell'elemento con **ID "Nota"**.  
  
11\. Successivamente, **fai clic su** sul **selettore XPath corrispondente " /html/body"**.  
  
12\. Alla fine **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pCdDcons"** .  
  
13\. Alla fine **Clicca** sull'elemento **"Salva Ed Esci"**.
