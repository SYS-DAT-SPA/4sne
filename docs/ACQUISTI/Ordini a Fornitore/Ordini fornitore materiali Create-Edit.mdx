### Guide for Ordini fornitore materiali Create-Edit

*Add the description here*

___

1\. Inizia facendo **clic su** sull'elemento **+**  
  
2\. Ora, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pCdAna"** .  
  
3\. **Seleziona** un'opzione dall'elemento a discesa **con data test è uguale a "pCdTipo"** .  
  
4\. In secondo luogo, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pTipo"** .  
  
5\. Proprio in quel momento, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pCdStagion"** .  
  
6\. Successivamente, **fai clic su** sull'elemento **"Salva "**.  
  
7\. Poco dopo, **fai clic su** sull'elemento **"Nota "**.  
  
8\. Successivamente, **Digita** **"NOTA"** nell'elemento **casella di testo**.  
  
9\. Alla fine **Clicca** sull'elemento **"Salva "**.
