Classe anagrafica

1.Classe Anagrafica - Guida all'Uso​

La Classe Anagrafica in 4S Next rappresenta una categorizzazione specifica applicabile ad elementi anagrafici come clienti, fornitori, o articoli. Questa classificazione consente di organizzare e segmentare i dati per analisi, reportistica o altre operazioni gestionali.

Esempi pratici

  1. Segmentazione dei clienti: Creare una classe anagrafica per distinguere i clienti premium da quelli standard.
  2. Gestione degli articoli: Utilizzare classi anagrafiche per categorizzare articoli per stagionalità o linea di produzione.
  3. Monitoraggio dei fornitori: Identificare i fornitori con una classe anagrafica specifica, ad esempio "Fornitore locale" o "Fornitore estero".

2. Guida Step-by-Step per 4S Next Edition​

Creazione di una Classe Anagrafica​

  1. Avvia la creazione
    • Clicca sull'elemento con dati test "+" per iniziare la creazione di una nuova classe anagrafica.
  2. Compila i campi obbligatori
    • Codice: Digita il codice identificativo della classe.
    • Descrizione: Inserisci una descrizione significativa (es. "Clienti Premium").
  3. Inserimento della data
    • Digita la data di riferimento nel formato richiesto, ad esempio "22/01/2023" nel campo "Data di Trasferimento".
  4. Definisci la co-classe
    • Inserisci un identificativo aggiuntivo per classificazioni avanzate (es. "I" per interni).
  5. Salvataggio iniziale
    • Clicca su "Salva" per confermare i dati inseriti.
  6. Gestione della tabella
    • Fai clic sulla cella di intestazione per inserire eventuali modifiche o personalizzazioni nella tabella.
    • Digita il nuovo valore nel campo di input associato alla cella.
  7. Modifica dei dati esistenti
    • Seleziona una riga della tabella e clicca sull'icona "Modifica" per modificare.
    • Apporta le modifiche necessarie e clicca su "Salva".
  8. Ricerca della classe anagrafica
    • Utilizza la funzione Cerca inserendo il codice o la descrizione per individuare l'elemento desiderato.
  9. Eliminazione
    • Individua l'elemento da eliminare tramite la ricerca, quindi clicca su "Elimina" e conferma l'operazione.

3.FAQ​

  1. Cos'è una classe anagrafica? Una classe anagrafica è una categorizzazione utilizzata per classificare clienti, fornitori, articoli o altri elementi gestionali in base a criteri specifici.
  2. Quali sono i campi obbligatori per creare una classe anagrafica? Sono obbligatori: il codice, la descrizione e, se richiesto, una data di riferimento.
  3. Come si utilizza la co-classe? La co-classe è un campo opzionale che può essere usato per aggiungere un ulteriore livello di categorizzazione, ad esempio per distinguere articoli interni da quelli esterni.
  4. Posso modificare una classe anagrafica esistente? Sì, basta selezionare l'elemento desiderato tramite la funzione Cerca, modificarlo con modifica, e salvare le modifiche.
  5. Cosa succede se elimino una classe anagrafica? La classe sarà rimossa dal sistema, ma le informazioni già collegate ad essa (come ordini o articoli) rimarranno in archivio senza il riferimento alla classe.
  6. Posso collegare più entità alla stessa classe? Certamente. Una classe può essere utilizzata per raggruppare più clienti, articoli o fornitori con caratteristiche simili.
  7. Cosa rappresenta il campo Data di Trasferimento? Indica la data di riferimento per la classe, utile per applicazioni temporali o reportistica specifica.
