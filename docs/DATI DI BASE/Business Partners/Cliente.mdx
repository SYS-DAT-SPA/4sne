2\. Per prima cosa **fai clic su** sull'elemento **con dati test uguale a a "RagSoc"**.  
  
3\. Subito dopo, **Digitare** **"Pluto 1 srl "** nell'elemento **con dati test uguale a "RagSoc"**.  
  
4\. Successivamente, **fai clic su** sull'elemento **con test dei dati equivale a "Indirizzo"**.  
  
5\. Quindi, **Digita** **"via Roma"** nell'elemento **con dati test uguale a "Indirizzo"**.  
  
6\. Quindi, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pCdNazione"** .  
  
7\. Proprio in quel momento, **Digita** **"ita"** nell'elemento **input field.select2 search\_\_field**.  
  
8\. Successivamente, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pCdProv"** .  
  
9\. Successivamente, **Digita** **"mi"** nell'elemento **input field.select2 search\_\_field**.  
  
10\. Poco dopo, **fai clic su** sull'elemento **span. select2 search.select2 menu a discesa di ricerca**.  
  
11\. Subito dopo, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pAllegati"** .  
  
12\. Successivamente, **fai clic su** sull'elemento **con i dati test equivale a "Piva"**.  
  
13\. Poco dopo, **Digita** **"59693790046"** nell'elemento **con dati test uguale a "Piva"**.  
  
14\. Successivamente, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pCdLingua"** .

19\. Successivamente, **fai clic su** sull'elemento **"Ok "**.  
  
20\. Successivamente, **Digita** **"00861"** nell'elemento **con segnaposto uguale a "Codice"**.  
  
21\. Poco dopo, **fai clic su** sull'elemento **i. far**.  
  
22\. Immediatamente dopo, **fai clic su** sull'elemento **con i dati test equivale a "Indirizzo"**.
