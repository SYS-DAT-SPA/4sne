Esercizi Test Guide

Guide Tutorial for Esercizi Test
================================

Step 1: Use Nuovo
-----------------

Click on the "Nuovo" button to start creating a new entry.

![](https://img.icons8.com/plumpy/24/select-none.png)

Step 2: Fill in Codice and Descrizione
--------------------------------------

Enter "2049" in the Codice field and "INTEST" in the Descrizione field.

![](https://img.icons8.com/plumpy/24/select-none.png) ![](https://img.icons8.com/plumpy/24/select-none.png)

Step 3: Select Azienda and Tipo
-------------------------------

Choose the options for Azienda and Tipo from the dropdown menus.

![](https://img.icons8.com/plumpy/24/select-none.png) ![](https://img.icons8.com/plumpy/24/select-none.png)

Step 4: Enter Dates
-------------------

Fill in the DataIniz, DataFine, and DataIniz fields with "22/01/2024".

![](https://img.icons8.com/plumpy/24/select-none.png)

Step 5: Enter Ultimo numero
---------------------------

Input "1" into the Ultimo numero field.

![](https://img.icons8.com/plumpy/24/select-none.png)

Step 6: Save the Entry
----------------------

Click on the "Salva" button to save the entry.

![](https://img.icons8.com/plumpy/24/select-none.png)

Step 7: Search for Entry
------------------------

Enter "2049" in the search field to find the entry.

![](https://img.icons8.com/plumpy/24/select-none.png)

Step 8: Add New Entry
---------------------

Click on the plus icon to add a new entry.

![](https://img.icons8.com/plumpy/24/select-none.png)

Step 9: Edit Entry
------------------

Click on the edit icon to make changes to the entry.

![](https://img.icons8.com/plumpy/24/select-none.png)

Step 10: Delete Entry
---------------------

Use the Elimina function to delete the entry.

![](https://img.icons8.com/plumpy/24/select-none.png)
