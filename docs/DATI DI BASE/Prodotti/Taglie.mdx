### Guide for Taglie

*Add the description here*

___

1\. Per prima cosa **fai clic su** sull'elemento **i.fa. fa-plus**.  
  
2\. Per prima cosa **fai clic su** sull'elemento **con dati test uguale a a "CdTaglia"**.  
  
3\. **Tipo** **"INTEST"** nell'elemento **con dati test uguale a "CdTaglia"** .  
  
4\. Quindi, **fai clic su** sull'elemento **con test dei dati equivale a "Descrizione"**.  
  
5\. Quindi, **Digita** **"taglie donna"** nell'elemento **con dati test uguale a "Descrizion"**.  
  
6\. Successivamente, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pTipotaglia"** .  
  
7\. Poco dopo, **fai clic su** sull'elemento **con i dati test è uguale a "Destg1"**.  
  
8\. Successivamente, **Digita** **"40"** nell'elemento **con dati test uguale a "Destg1"**.  
  
9\. Immediatamente dopo, **fai clic su** sull'elemento **con i dati test è uguale a "Destg2"**.  
  
10\. Successivamente, **Digita** **"42"** nell'elemento **con dati test uguale a "Destg2"**.  
  
11\. Ora, **fai clic su** sull'elemento **con test dei dati equivale a "Destg3"**.  
  
12\. Successivamente, **Digita** **"44"** nell'elemento **con dati test uguale a "Destg3"**.  
  
13\. Successivamente, **fai clic su** sull'elemento **con test dei dati equivale a "Destg4"**.  
  
14\. **Tipo** **"46"** nell'elemento **con test dei dati uguale a "Destg4"** .  
  
15\. Successivamente, **fai clic su** sull'elemento **con i dati test equivale a "Destg5"**.  
  
16\. Successivamente, **Digita** **"48"** nell'elemento **con dati test uguale a "Destg5"**.  
  
17\. In secondo luogo, **fai clic su** sull'elemento **con test dei dati equivale a "Destg6"**.  
  
18\. Successivamente, **Digita** **"50"** nell'elemento **con dati test uguale a "Destg6"**.  
  
19\. **Clicca** sull'elemento **"Fasce prezzo"**.  
  
20\. Proprio in quel momento, **Seleziona** un'opzione dal menu a discesa corrispondente a **Selettore XPath "/html/body"** .  
  
21\. In secondo luogo, **Seleziona** un'opzione dal menu a discesa corrispondente a **Selettore XPath "/html/body"** .  
  
22\. Successivamente, **fai clic su** sull'elemento **"Salva "**.  
  
23\. Alla fine var cerca = "INTEST"  
  
24\. Alla fine **Esegui i passaggi** per **[Cerca](/docs/Z_Operazioni%20base/Cerca)**  
  
25\. Alla fine **Esegui i passaggi** per **[Elimina](/docs/Z_Operazioni%20base/Elimina)**
