___

1\. Per prima cosa **fai clic su** sull'elemento **i.fa. fa-plus**.  
  
2\. Per prima cosa **fai clic su** sull'elemento **con dati test uguale a a "CdTaglia"**.  
  
3\. Proprio in quel momento, **Digita** **"INTEST"** nell'elemento **con dati test uguale a "CdTaglia"**.  
  
4\. Immediatamente dopo, **fai clic su** sull'elemento **con i dati test è uguale a "Descrizion"**.  
  
5\. Successivamente, **Digitare** **"taglie donna"** nell'elemento **con dati test uguale a "Descrizion"**.  
  
6\. Successivamente, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pTipotaglia"** .  
  
7\. **Fai clic su** sull'elemento **con i dati test uguali a "Destg1"**.  
  
8\. Successivamente, **Digita** **"40"** nell'elemento **con dati test uguale a "Destg1"**.  
  
9\. **Fai clic su** sull'elemento **con i dati test uguali a "Destg2"**.  
  
10\. Successivamente, **Digita** **"42"** nell'elemento **con dati test uguale a "Destg2"**.  
  
11\. Successivamente, **fai clic su** sull'elemento **con i dati test equivale a "Destg3"**.  
  
12\. Immediatamente dopo, **Digita** **"44"** nell'elemento **con dati test uguale a "Destg3"**.  
  
13\. Poco dopo, **fai clic su** sull'elemento **con i dati test è uguale a "Destg4"**.  
  
14\. Proprio in quel momento, **Digita** **"46"** nell'elemento **con dati test uguale a "Destg4"**.  
  
15\. In secondo luogo, **fai clic su** sull'elemento **con test dei dati equivale a "Destg5"**.  
  
16\. **Tipo** **"48"** nell'elemento **con test dei dati uguale a "Destg5"** .  
  
17\. Successivamente, **fai clic su** sull'elemento **con i dati test è uguale a "Destg6"**.  
  
18\. Successivamente, **Digita** **"50"** nell'elemento **con dati test uguale a "Destg6"**.  
  
19\. Subito dopo, **Click** sull'elemento **"Fasce prezzo"**.  
  
20\. Quindi, **Seleziona** un'opzione dal menu a discesa corrispondente a **Selettore XPath "/html/body"** .  
  
21\. Quindi, **Seleziona** un'opzione dal menu a discesa corrispondente a **Selettore XPath "/html/body"** .
