### Guide for Tipi materiale

*Add the description here*

___

1\. Per prima cosa **fai clic su** sull'elemento **con dati test uguale a in "Nuovo"**.  
  
2\. Proprio in quel momento, **fai clic su** sull'elemento **con i dati test è uguale a "CdTipMat"**.  
  
3\. Proprio in quel momento, **Digita** **"INTEST"** nell'elemento **con dati test uguale a "CdTipMat"**.  
  
4\. Successivamente, **fai clic su** sull'elemento **con i dati test è uguale a "Descrizion"**.  
  
5\. Successivamente, **Digita** **"tessuto"** nell'elemento **con dati test uguale a "Descrizion"**.  
  
6\. In secondo luogo, **Seleziona** un'opzione dall'elemento a discesa **con dati test uguale a "pCdMfase"** .  
  
7\. **Clicca** sull'elemento **"Materiale di Taglio"** .  
  
8\. Quindi, **Fai clic su** sul **selettore XPath corrispondente " /html/body"**.  
  
9\. Successivamente, **fai clic su** sul **selettore XPath corrispondente "/html/body"**.  
  
10\. Successivamente, **fai clic su** sul **selettore XPath corrispondente " /html/body"**.  
  
11\. Alla fine **Fai clic su** sul **selettore XPath corrispondente "/html/body"**.  
  
12\. Alla fine **Clicca** sull'elemento **"Salva "**.
