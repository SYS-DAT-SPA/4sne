### Guide for Classi materiali

*Add the description here*

___

1\. Proprio in quel momento, **fai clic su** sull'elemento **i. fa**.  
  
2\. Successivamente, **fai clic su** sull'elemento **con test dei dati equivale a "CdClsMat"**.  
  
3\. In secondo luogo, **Tipo** **"INTEST"** nell'elemento **con dati test uguale a "CdClsMat"**.  
  
4\. Quindi, **fai clic su** sull'elemento **con test dei dati equivale a "Descrizione"**.  
  
5\. Poco dopo, **Digita** **"intest"** nell'elemento **con dati test uguale a "Descrizion"**.  
  
6\. Quindi, **fai clic su** sull'elemento **i.far 2**.  
  
7\. Subito dopo, var cerca = "INTEST"  
  
8\. A quel punto, **Esegui i passaggi** per **[Cerca](/docs/Z_Operazioni%20base/Cerca)**  
  
9\. Alla fine **Esegui i passaggi** per **[Elimina](/docs/Z_Operazioni%20base/Elimina)**
