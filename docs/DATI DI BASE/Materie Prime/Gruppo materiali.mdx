### Guide for Gruppo materiali

*Add the description here*

___

1\. Proprio in quel momento, **fai clic su** sull'elemento **i. fa**.  
  
2\. Successivamente, **fai clic su** sull'elemento **con test dei dati equivale a "CdGruMat"**.  
  
3\. Successivamente, **Digita** **"INTEST"** nell'elemento **con dati test uguale a "CdGruMat"**.  
  
4\. Successivamente, **fai clic su** sull'elemento **con test dei dati equivale a "Descrizione"**.  
  
5\. Successivamente, **Digita** **"intest"** nell'elemento **con dati test uguale a "Descrizion"**.  
  
6\. Ora, **fai clic su** sull'elemento **"Salva"** .  
  
7\. Successivamente, var cerca = "INTEST"  
  
8\. Successivamente, **Esegui i passaggi** per **[Cerca](../docs/Z_Operazioni%20base/Cerca)**  
  
9\. Alla fine **Esegui i passaggi** per **[Elimina](../docs/Z_Operazioni%20base/Elimina)**
